When the grid is initially created, it is set by default to one column.

        
            .grid-container1 {
                display: grid;
            }

We can add columns to make it look more grid-like.

        
            .grid-container2 {
                display: grid;
                grid-template-columns: 100px 100px 100px;
            }

To create a flexible grid, we can use the fr unit, which will resize the columns to fit proportionally based on the numbers given. The left column now has most of the available space.

        
            .grid-container3 {
                display: grid;
                grid-template-columns: 2fr 1fr 1fr;
            }

We can also add gaps using row-gap and column-gap for rows and columns respectively. gap is used to affect both.

            
                .grid-container4 {
                    display: grid;
                    grid-template-columns: 2fr 1fr 1fr;
                    row-gap: 10px;
                    column-gap: 20px;
                }
