CSS Grid Layout Overview

When the grid is initially created, it is set by default to one column.

        
            .grid-container {
                display: grid;
            }

We can add columns to make it look more grid-like.

        
            .grid-container {
                display: grid;
                grid-template-columns: 100px 100px 100px;

To create a flexible grid, we can use the fr unit, which will resize the columns to fit proportionally based on the numbers given. The left column now has most of the available space.

        
            .grid-container {
                display: grid;
                grid-template-columns: 2fr 1fr 1fr;
            }

We can also add gaps using row-gap and column-gap for rows and columns respectively. gap is used to affect both.

            
                .grid-container {
                    display: grid;
                    grid-template-columns: 2fr 1fr 1fr;
                    row-gap: 10px;
                    column-gap: 20px;
                }

Many different functions can be combined for more complex patterns. Using the repeat() function to create multiple columns at once, the minmax() function to set a minimum size for columns, and auto-fill to add a flexible number of columns, we can add as many columns as can fit on the given page.

            
                .grid-container {
                    display: grid;
                    grid-template-columns: repeat(auto-fill, 400px);
                    grid-auto-rows: 100px, auto;
                }

More notes on Grid Layout:

  • Implicit and explicit grids: The explicit grid is defined using -template properties, like grid-template-columns, and the implicit is defined using -auto properties, like grid-auto-columns.
  • Line-based placement: The lines in a grid start at 1, and can be specified with properties such as grid-column-start, grid-column-end, grid-row-start, and grid-row-end.
  • Grid template areas: By renaming different elements of the grid with names like header, sidebar, or footer, the need for line numbers is eliminated.
